# Weather App

A simple weather application that uses Open-Meteo APIs to look up current weather conditions for one or more cities.

## Features

- Search for single or multiple cities
- Fetch city coordinates from Open-Meteo Geocoding API
- Get current weather and humidity from Open-Meteo Forecast API
- Show temperature, wind speed, humidity, and weather condition
- Handle city lookup and weather request errors gracefully

## Run locally

Open `index.html` in your browser, or serve the folder with a static file server.

## Notes

No API keys are required. The app uses free Open-Meteo endpoints directly from the browser.
